# Contributing

Thanks for helping improve this list.

## How To Contribute

Open a pull request with your proposed change.

## What Is Accepted

Add tools that are either:
- Self-hostable options
- EU companies

Not accepted:
- US companies or other non European companies, even if they have EU branches

## Quality Criteria

Only suggest tools you have actually tried and have real experience with.

In your PR description, briefly include:
- How you used the tool
- Why it belongs in this list
- What infra they run on

## Code Of Conduct

Keep communication respectful and constructive.

## Maintenance Notes (Dead Links)

If you find a dead link:
- Open a PR to fix the URL, replace the tool, or remove the entry
- Add a short note in the PR describing what you checked
